Instances API

Manage compute resources.

Provision, scale, and inspect virtual instances across the global edge network.

Endpoint
GET /v1/instances
Authentication

All calls require a bearer token set in the Authorization header. Use API keys from your account settings.

View Example
Request example
curl -X POST https://api.domainc.com/v1/instances \ -H "Authorization: Bearer $API_KEY" \ -H "Content-Type: application/json" \ -d '{ "plan_id": "std-2-4", "region": "us-east-1", "image": "ubuntu-22-04-lts", "label": "production-db-01" }'
Response
{ "status": "success", "data": { "id": "inst_928kL82p", "status": "provisioning", "ipv4": "157.230.12.44", "created_at": "2023-10-27T14:20:01Z", "config": { "plan": "std-2-4", "region": "us-east-1" } } }
